# Tennis Masters 벤치마크 기록

대상: [Tennis Masters by Madpuffers](https://poki.com/kr/g/tennis-masters)

## 확인한 특징

- 일반 타격과 스매시가 분리된 간단한 조작 구조
- 캐릭터보다 낮은 네트와 넓은 타격 여유
- 공의 수평 이동은 안정적이고 중력에 따른 상승·낙하는 부드러움
- 네트 상단을 스친 공이 속도를 잃고 가까운 코트에 떨어지는 장면이 자주 발생
- 캐릭터는 즉시 방향을 바꾸지 않고 짧은 가감속을 거침

## 구현 추론

내부 소스는 공개되어 있지 않으므로 다음 내용은 플레이 결과를 기반으로 한 추론이다.

1. 일반 타격은 완전한 현실 물리보다 목표 궤적을 보정하는 아케이드 방식에 가깝다.
2. 네트 몸통과 상단 끈의 충돌 반응을 다르게 처리하면 자연스러운 네트 드롭을 만들 수 있다.
3. 공 전체 속도를 매 프레임 고정하기보다 수평속도를 유지하고 상승·낙하 속도만 제한하는 편이 자연스럽다.
4. 이동 최고속도 배수보다 낮은 최고속도와 부드러운 가감속이 화면을 읽기 쉽게 만든다.

## CatPong 반영

- 네트 총 높이 `1.05`, 상단 끈 높이 `0.05`
- `NetCord` 전용 원형 콜라이더와 접촉 감쇠
- 짧은 J 입력 시 현재 공 위치에서 네트 위 `0.07`을 통과하는 궤적 역산
- 공 수평·상승·낙하 속도 개별 제한
- 플레이어와 AI의 숨은 속도 배수 제거, 부드러운 가감속 적용
