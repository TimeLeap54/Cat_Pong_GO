# 게임 기획서 v0.1

## 프로젝트명

Cat Tennis Tournament / 냥냥 테니스 토너먼트

## 개발 목표

2D 사이드뷰 테니스 게임을 7~14일 안에 WebGL로 배포한다. 목표는 대형 상업 게임이 아니라 Unity + Codex + AI 이미지 에셋 파이프라인을 검증하고 첫 배포 경험을 얻는 것이다.

## 핵심 콘셉트

고양이 캐릭터가 AI 상대들과 2D 테니스 토너먼트를 진행하는 짧은 아케이드 스포츠 게임.

## 핵심 게임 루프

1. 메인 메뉴에서 토너먼트 시작
2. AI 상대와 5점제 경기 진행
3. 승리하면 다음 라운드 진출
4. 패배하면 게임오버
5. 3라운드 결승까지 이기면 토너먼트 우승
6. 다시하기로 재도전

## 기본 조작

- `A/D` 또는 방향키: 좌우 이동
- `Space`: 점프
- `J` 또는 마우스 클릭: 스윙

## 경기 규칙

- 플레이어는 왼쪽 코트, AI는 오른쪽 코트에 위치한다.
- 공이 자기 코트 바닥에 닿으면 상대가 점수를 얻는다.
- 먼저 5점을 얻은 쪽이 승리한다.
- 현실 테니스의 서브, 듀스, 아웃라인, 세트 규칙은 v0.1에서 제외한다.
- 네트에 공이 닿으면 튕기거나 약하게 감속된다.
- 공이 화면 밖으로 나가면 마지막으로 공을 넘기지 못한 쪽이 실점한다.

## 토너먼트 구조

| Round | Opponent | 목표 |
| --- | --- | --- |
| 1 | Rookie Cat | 처음 하는 유저도 쉽게 승리 |
| 2 | Dojo Cat | 접전 발생 |
| 3 | Master Cat | 어렵지만 이길 수 있는 결승전 |

## 성공 기준

- itch.io에 WebGL 배포 완료
- 링크를 통해 다른 사람이 플레이 가능
- 3라운드 토너먼트가 처음부터 끝까지 진행됨
- 승리/패배/재시작이 정상 작동
