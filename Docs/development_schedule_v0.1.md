# 개발 일정표 v0.1

## 전체 목표

- 7일 안에 WebGL로 1차 배포한다.
- 14일 안에 피드백을 반영한 개선판을 배포한다.

## 개발 원칙

- 처음 2일은 에셋 작업 금지
- 사각형과 원으로 먼저 플레이 가능한 게임을 만든다
- 토너먼트가 돌아가기 전까지 기능 추가 금지
- 매일 마지막 1시간은 테스트와 기록에 사용한다

## Day 1: 기본 경기 뼈대

목표: 공, 플레이어, AI 위치가 있는 경기 화면 만들기

작업:

- Unity 2D 프로젝트 생성
- 폴더 구조 생성
- Match 씬 생성
- 코트, 네트, 플레이어, AI, 공 배치
- `PlayerController.cs` 생성
- `BallController.cs` 생성
- 플레이어 좌우 이동 구현
- 공 `Rigidbody2D` 이동 구현
- 벽/바닥/네트 Collider 설정

완료 기준:

- 플레이어가 움직인다
- 공이 화면 안에서 움직인다
- 공이 벽/바닥/네트와 충돌한다

## Day 2: 스윙과 득점

목표: 단판 경기가 가능한 상태 만들기

작업:

- 점프 구현
- 스윙 입력 구현
- 스윙 범위 안에 공이 있으면 공 반사
- 좌우 코트 바닥 충돌 시 득점 판정
- `MatchManager.cs` 생성
- 5점제 승리/패배 구현
- 공 리셋 구현
- 점수 UI 텍스트 연결

완료 기준:

- 플레이어가 공을 칠 수 있다
- 점수가 오른다
- 5점 먼저 얻으면 승패가 나온다

## Day 3: AI 상대와 토너먼트

목표: 3명의 AI를 순서대로 상대하는 구조 만들기

작업:

- `OpponentAI.cs` 생성
- AI가 공 위치를 따라 이동하도록 구현
- AI 스윙 판정 구현
- AI 난이도 파라미터 추가
- `TournamentManager.cs` 생성
- Round 1, Round 2, Final 구성
- 승리 시 다음 라운드 이동
- 패배 시 게임오버
- 결승 승리 시 우승 화면

완료 기준:

- AI와 경기가 가능하다
- 이기면 다음 상대가 나온다
- 3명을 이기면 우승한다

## Day 4: 물리 오류 수정과 조작감 조정

목표: 억울한 판정과 이상한 공 움직임 줄이기

작업:

- 공 속도 제한
- 공이 네트/바닥에 끼는 문제 수정
- 득점 중복 발생 방지
- 공이 화면 밖으로 나가는 경우 처리
- 스윙 범위 조정
- 스윙 쿨타임 추가
- 점프 높이와 이동속도 조정
- 공 반사 각도 조정

완료 기준:

- 5판 연속 플레이해도 치명적 버그가 없다
- 공을 치는 감각이 최소한 납득 가능하다

## Day 5: AI 밸런싱

목표: 3라운드 난이도 차이가 느껴지게 만들기

작업:

- Round 1 AI 수치 조정
- Round 2 AI 수치 조정
- Final AI 수치 조정
- `moveSpeed`, `reactionDelay`, `mistakeRate` 조정
- 테스트 경기 기록
- 평균 점수차 확인
- 너무 쉬운/너무 어려운 구간 수정

목표 감각:

- Round 1: 쉽게 이김
- Round 2: 가끔 위험함
- Final: 집중해야 이김

완료 기준:

- 세 AI의 난이도 차이가 확실히 느껴진다
- 결승이 불가능하거나 억울하지 않다

## Day 6: UI / 사운드 / 에셋 교체

목표: 최소한 남에게 보여줄 수 있는 형태로 만들기

작업:

- MainMenu 씬 생성
- Start 버튼 구현
- Restart 버튼 구현
- Win/Lose 화면 구현
- Round 표시 UI 추가
- AI 이름 표시
- 고양이 캐릭터 이미지 삽입
- 코트 배경 이미지 삽입
- 공/네트 이미지 삽입
- 공 타격음 추가
- 득점음 추가
- 버튼 클릭음 추가
- BGM 1개 추가

완료 기준:

- 사각형 게임이 아니라 간단한 완성 게임처럼 보인다
- 시작/경기/결과/재시작 흐름이 자연스럽다

## 14일 후 판단 기준

계속 개발할 가치 있음:

- 플레이한 사람이 다시 하려고 함
- 조작감이 재밌다는 반응이 있음
- AI 토너먼트 구조가 괜찮다는 반응이 있음

버리고 다음으로 가도 됨:

- 조작 자체가 재미없음
- 고쳐도 공 물리가 계속 불쾌함
- 유저 반응이 거의 없음
- 본인이 더 만들고 싶지 않음
